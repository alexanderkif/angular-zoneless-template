About This Project

This is a modern Angular 20 application template featuring zoneless change detection, server-side rendering (SSR), and a robust authentication system.

Key Features

  • Angular 20.3: Leveraging the latest stable version with zoneless architecture for superior performance.
  • Server-Side Rendering (SSR): Powered by Angular Universal for improved SEO and initial load times.
  • Secure Authentication: Email & OAuth (GitHub, Google) integration using Supabase and JWT.
  • State Management: Implemented with NgRx Store and Effects for predictable data flow.
  • Comprehensive Testing: 100% test coverage with Vitest and End-to-End testing with Playwright.

Tech Stack

Core

Angular 20.3, TypeScript 5.9, RxJS 7.8, Express 5.1

State

NgRx Store, Effects, DevTools

Backend & Auth

Supabase, Vercel Functions, JWT, Argon2id

Testing

Vitest, Playwright, jsdom
