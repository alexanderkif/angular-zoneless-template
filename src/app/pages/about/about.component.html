About This Project

This is a modern Angular 21 application template featuring zoneless change detection, server-side rendering (SSR), and a robust authentication system.

Key Features

  • Angular 21 + Signals: Leveraging the latest stable version with zoneless architecture, Signals-based reactivity, and functional patterns.
  • Server-Side Rendering (SSR): Optimized hydration with Event Replay for a seamless user experience from initial load.
  • Secure Authentication: Multi-provider OAuth (GitHub, Google) and Email/Password with HttpOnly cookie rotation and CSRF protection.
  • State Management: Modern NgRx with Action Groups and selectSignal for reactive state bridging.
  • Performance First: Optimized image loading (NgOptimizedImage) and standardized accessibility (WCAG compliance).

Tech Stack

Core

Angular 21, TypeScript 5.9, RxJS 7.8, Vite 7.3, ESM

State & Data

NgRx 21, Signals, Signal Forms, Drizzle ORM

Backend & Auth

Node.js, Supabase, Vercel Functions, JWT, Argon2id, Zod

Testing

Vitest, Playwright, 100% Coverage Target
